2. Configuración básica por modos

Modo Usuario

Este es el modo inicial al acceder al switch. Permite comandos básicos de verificación.

ComandoDescripción
show ip interface briefMuestra un resumen de las interfaces y su estado.
ping <IP>Realiza pruebas de conectividad.
?Muestra una lista de comandos disponibles.

Para acceder al modo privilegiado:

Modo Privilegiado

Permite realizar configuraciones avanzadas y verificar el estado del switch.

ComandoDescripción
show vlan briefMuestra información sobre las VLAN configuradas.
show mac address-tableMuestra la tabla de direcciones MAC.
?Muestra una lista de comandos disponibles.

Para acceder al modo de configuración global:

Modo de Configuración Global

Permite realizar configuraciones en todo el switch o en componentes específicos.

ComandoDescripción
enable secret <CONTRASEÑA>Configura una contraseña encriptada para el modo privilegiado.
?Muestra una lista de comandos disponibles.
